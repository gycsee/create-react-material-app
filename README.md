# React + Material-UI

> 基于 React 和 Material-UI 的中后台应用模版

## Usage

```sh
# Clone
$ git clone --depth=0 https://github.com/gycsee/create-react-material-app.git

# Setup
$ npm install

# Start dev server
$ npm start

# Build for production
$ npm build
```
